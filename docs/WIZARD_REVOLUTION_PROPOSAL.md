## Executive Summary

### The Problem

The current wizard suffers from:
- **Step 1 Location = Friction Monster**: Map requires search, zoom (5+ taps on mobile!), click to pin
- **Wrong Order**: We ask for location BEFORE capturing any lead data
- **High Drop-off**: Highest abandonment at Step 1 - users quit before we know who they are
- **Mobile Suffering**: 400px map on small screens, tiny touch targets, complex interactions

### The Solution

Transform a 7-step friction-heavy wizard into a 6-step conversion machine:

```
OLD: Location â†’ Consumption â†’ System â†’ Financing â†’ Bill â†’ Contact â†’ Summary
NEW: Region â†’ Calculator â†’ Safety â†’ Contact â†’ System â†’ Location
```

| Metric | Current | Target | Moonshot |
|--------|---------|--------|----------|
| Lead Capture | ~15% | 40% | **60%** |
| Time to Lead | 4+ min | 90 sec | **60 sec** |
| Mobile Completion | ~40% | 70% | **85%** |
| Extras Upsell | 0% | 30% | **50%** |

### Design Specifications

**Layout**
- Full viewport height (100dvh) - no scrolling on mobile
- Cards use golden ratio proportions (1:1.618)
- 24px padding on mobile, 48px on desktop
- Cards are 47% width each on desktop, stacked on mobile

**Typography**
- Hero: "Your Solar Journey" - 32px mobile / 48px desktop, font-weight 700
- Subhead: 18px, font-weight 400, opacity 70%
- Card titles: 24px, font-weight 600
- Grant amounts: 16px, font-weight 500

**Colors**
- Background: Deep navy gradient (#0a0a0f â†’ #0d1117)
- Gozo card: Subtle amber glow on hover (box-shadow: 0 0 60px rgba(245,158,11,0.15))
- Malta card: Subtle blue glow on hover
- Enhanced badge: Amber gradient background
- Text: White primary, gray-400 secondary

**Imagery**
- Photos: 16:10 aspect ratio, object-fit: cover
- Subtle Ken Burns effect (slow zoom) on hover
- Overlay gradient from bottom (for text legibility)
- Consider: commission drone footage of actual GhawdeX installations

**Animations**


```typescript
function calculateInstantQuote(monthlyBill: number, location: 'malta' | 'gozo', isBatteryOnly: boolean) {
  // Convert bill to estimated consumption
  const annualBill = monthlyBill * 12;
  const estimatedKwh = annualBill / 0.16; // â‚¬0.16/kWh average

  // Recommend system size (90% coverage target)
  const recommendedKwp = Math.ceil(estimatedKwh / 1500); // ~1500 kWh per kWp in Malta
  const system = SYSTEMS.find(s => s.sizeKwp >= recommendedKwp) || SYSTEMS[SYSTEMS.length - 1];

  // Calculate grant (simplified for instant quote)
  const pvGrant = Math.min(system.basePrice * 0.5, 3000);

  // Annual savings (simplified)
  const annualProduction = system.annualProductionKwh;
  const selfConsumptionRatio = 0.7;
  const annualSavings = annualProduction * selfConsumptionRatio * 0.16;

  // Payback
  const netCost = system.basePrice - pvGrant;
  const paybackYears = netCost / annualSavings;

  return {
    system,
    grossPrice: system.basePrice,
    grant: pvGrant,
    netPrice: netCost,
    annualSavings,
    paybackYears,
    lifetimeSavings: annualSavings * 25,
  };
}
```

### Copywriting Principles
- Never say "upsell" - it's "safety equipment"
- Never say "optional" for safety items - say "recommended"
- Frame as "we've got you covered" not "you need to buy"
- Use "your family" and "your investment" - personal stakes
- Bundle: Don't just show savings, show extra VALUE (warranty, new board)

### Design Specifications

**Form Fields**
- Height: 56px (exceeds touch target requirements)
- Border radius: 12px
- Background: rgba(255,255,255,0.05)
- Focus state: Amber border, subtle glow
- Label: Animated float-up on focus (like Material Design)

**Phone Input**
- Prefix: Non-editable +356 in muted color
- Auto-format: 9999 9999 pattern
- Keyboard: tel type for numeric keypad

**Email Input**
- Keyboard: email type
- Real-time validation with subtle âœ“ or âœ—

**Submit Button**
- Full width
- Height: 56px
- Gradient: Amber to Orange
- Hover: Slight lift + shadow increase
- Loading state: Subtle pulse + "Saving..." text
- Success state: Green + checkmark + "Saved!"

**Social Proof Section**
- Avatar stack: Overlapping circles with faces
- Counter: Animated counting number
- Copy: "Join X families who've made the switch"

**CRITICAL**: This is where the lead is created. Even if they abandon after this, we have their contact info and all their preferences from Steps 1-3.

### Design Specifications

**Map**
- Simplified - zooms to selected island automatically (Malta or Gozo from Step 1)
- "Find Me" button prominent
- Simpler pin placement (allow without max zoom requirement)
- Or: Make map OPTIONAL with "Skip, contact me to arrange site visit"

**Quote Summary**
- Celebratory header with emoji
- Clear line items with icons
- Grant percentage called out ("75% covered!")
- Payback and lifetime savings highlighted

**Action Buttons**
- Primary: Download PDF (amber)
- Secondary: Email Quote (outline)
- Tertiary: Call Me Now (green, for high-intent users)

**Completion Celebration**
- Confetti animation when they pin location (subtle, not annoying)
- Success message: "ğŸ‰ Your quote is ready!"
- Testimonial for social proof and reassurance

### Haptic Feedback (Mobile)

```typescript
const haptics = {
  light: () => navigator.vibrate?.(10),
  medium: () => navigator.vibrate?.(20),
  heavy: () => navigator.vibrate?.(30),
  success: () => navigator.vibrate?.([20, 50, 20]),
  error: () => navigator.vibrate?.([50, 30, 50]),
};

// Use on:
// - Button taps: light
// - Slider movement: light (throttled)
// - Selection made: medium
// - Form submitted: success
// - Validation error: error
```
