## What is Battery-Only Grant?

### Definition

The **Battery-Only Grant** (Option D in REWS scheme) is a government subsidy specifically for purchasing residential battery storage systems **without requiring a new PV installation**.

### Who Can Benefit?

1. **Existing Solar Owners:** Already have PV system, want to add battery
2. **Grid Arbitrage Users:** No solar, but want to charge battery at night (cheaper) and use during peak hours
3. **Backup Power Seekers:** Want battery for power outages (Malta has occasional grid issues)
4. **Future Solar Planners:** Buy battery now with grant, add solar later

### What's Included in "Battery-Only"

When a customer purchases a "battery-only" system, the price typically includes:

- Battery storage unit (5-15 kWh)
- Battery Management System (BMS)
- DC/AC inverter (hybrid or storage-specific)
- Installation and commissioning
- Grid connection modifications
- Monitoring system

**IMPORTANT:** The grant applies to the **total battery system cost** including any necessary inverter or hardware. There is NO separate "inverter grant" for battery-only purchases.

### What's NOT Included

- Solar panels (PV)
- Solar-specific inverter grants
- Feed-in tariff (no export capability without PV)

   - Installation by REWS-registered retailer/installer

   - Adequate space for battery installation

## Grant Calculation Methodology

### The Three-Way Minimum Rule

Battery grant is calculated as the **minimum of three values**:

```
Grant = min(
    kWh × €720,           // Per kWh rate cap
    Price × 80%/95%,      // Percentage of actual cost
    €7,200/€8,550         // Maximum cap
)
```

### Malta Calculation (80%)

```javascript
function calculateBatteryGrantMalta(batteryKwh: number, batteryPrice: number): number {
  const kwhBasedGrant = batteryKwh * 720;           // €720/kWh
  const percentageBasedGrant = batteryPrice * 0.80; // 80% of price
  const maxCap = 7200;                               // €7,200 cap

  return Math.min(kwhBasedGrant, percentageBasedGrant, maxCap);
}
```

### Gozo Calculation (95%)

```javascript
function calculateBatteryGrantGozo(batteryKwh: number, batteryPrice: number): number {
  const kwhBasedGrant = batteryKwh * 720;           // €720/kWh
  const percentageBasedGrant = batteryPrice * 0.95; // 95% of price
  const maxCap = 8550;                               // €8,550 cap

  return Math.min(kwhBasedGrant, percentageBasedGrant, maxCap);
}
```

## How Batteries Save Money (Without Solar)

### The Battery Arbitrage Strategy

Even without solar panels, batteries can save money through **tariff arbitrage**:

1. **Charge at night** when electricity is cheapest
2. **Use during day** when rates are highest
3. **Pocket the difference** as savings

### Malta's Tiered Tariff System

Malta uses progressive consumption-based tariffs (NOT time-of-use for most residential):

| Annual Band | kWh Range | Rate per kWh |
|-------------|-----------|--------------|
| Band 1 | 0 - 2,000 | €0.1047 |
| Band 2 | 2,001 - 6,000 | €0.1298 |
| Band 3 | 6,001 - 10,000 | €0.1607 |
| Band 4 | 10,001 - 20,000 | **€0.3420** |
| Band 5 | 20,001+ | **€0.6076** |

### The Marginal Rate Advantage

Battery savings are calculated at the **marginal rate** (highest tier customer is in):

```
High Consumer (15,000 kWh/year):
├─ Pays €0.3420/kWh for last kWh consumed (Band 4)
├─ Battery reduces consumption by ~1,282 kWh/year (10kWh battery)
├─ Each kWh saved = €0.3420 avoided
└─ Annual savings = 1,282 × €0.3420 = €438
```

### Battery-Only Grant Logic

```typescript
// From /src/lib/types.ts lines 290-310

} else if (grantType === 'battery_only') {
  // Battery-only grant (for customers who already have solar or want standalone battery)
  // Battery: 80% (Malta) or 95% (Gozo) of battery costs
  // Note: Battery price includes all necessary hardware (inverter, integration, etc.)
  // so we only apply the battery grant, not a separate inverter grant
  let batteryGrant = 0;
  if (batteryKwh && batteryKwh > 0) {
    const batteryKwhBasedGrant = batteryKwh * GRANT_SCHEME_2025.BATTERY[location].perKwh;
    const batteryPercentageBasedGrant = batteryPrice
      ? batteryPrice * GRANT_SCHEME_2025.BATTERY[location].percentage
      : batteryKwhBasedGrant;

    batteryGrant = Math.min(
      batteryKwhBasedGrant,
      batteryPercentageBasedGrant,
      GRANT_SCHEME_2025.BATTERY[location].maxTotal
    );
  }

  totalGrant = batteryGrant;
}
```

```typescript
// From /src/lib/calculations.ts lines 378-394

// Handle battery-only case (no solar system)
```

## Frequently Asked Questions

### General Questions

**Q: Can I get a battery without having solar panels?**
> Yes! The REWS Option D (Battery Only) is specifically designed for standalone battery purchases. You don't need PV to qualify.

**Q: Do I need to have an existing solar system?**
> No. Battery-only grants are available whether or not you have existing solar. However, if you have an old non-hybrid inverter and want to add battery, Option C may be better.

**Q: Can I get battery grant if I already received PV grant?**
> Yes, you can apply for battery grant even if you previously received a PV grant (as long as it wasn't within the last 6 years for the same property).

### Grant Calculation Questions

**Q: Why is the grant sometimes less than 80% of the price?**
> The grant is capped at €720/kWh. If your battery costs more than €900/kWh (for Malta), the percentage cap applies first. Example: €8,000 for 10kWh battery = €800/kWh. At 80%, grant is €6,400 (limited by percentage, not per-kWh cap).

**Q: Why do Gozo residents get more grant?**
> Government policy provides enhanced support for Gozo to promote energy independence. Gozo gets 95% coverage vs 80% for Malta, and a higher maximum cap (€8,550 vs €7,200).

**Q: Is there a separate inverter grant for battery-only?**
> No. Option D (Battery Only) includes just the battery grant. The battery system price already includes any necessary inverter hardware. If you need a new hybrid inverter, use Option C instead.

### Technical Questions

**Q: What battery brands are eligible?**
> Batteries must be from REWS-approved manufacturers. GhawdeX uses Huawei LUNA2000 series (LFP technology, 10-year warranty).

**Q: What's the minimum battery size?**
> Minimum 2.5 kWh capacity to qualify for grant.

**Q: Can I add more battery capacity later?**
> Yes, you can expand battery capacity later. However, additional units would need a new grant application (subject to availability and 6-year cooling-off period).

### Financial Questions

**Q: How long until I get the grant money?**
> Typically 4-6 weeks after Part B submission. The grant is paid directly to your bank account.

**Q: Do I pay full price upfront?**
> Yes, you pay the installer the full price. The grant is refunded to you after installation and Part B approval.

**Q: Can I finance the battery?**
> Yes! BOV Personal Energy Loan covers battery purchases at 0.5% interest for first 3 years. You can pay back the loan with the grant when it arrives.
